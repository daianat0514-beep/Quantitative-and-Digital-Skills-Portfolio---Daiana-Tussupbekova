This project analyzes the weekly stock returns of Apple and Microsoft and compares their performance and volatility. Historical price data were obtained from Yahoo Finance and converted into percentage returns for statistical analysis.
This analysis includes descriptive statistics, hypothesis testing (Chi-square and F-test), and CAMP regression to examine the relationship between Apple's returns and the S&P 500 market index.
The project demonstrates quantitative and digital data analysis, statistical testing, regression analysis, and data visualization using spreadsheet tools.
